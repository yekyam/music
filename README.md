# music

Simple music player command line utility.

## Features

Does what it says on the label.

- Add songs to a library.
- Randomly shuffles songs from the library.
- Repeat tracks
- Repeat entire library
- Skip
- Pause

## Pros

- Small
- Easy to use
- Works 

## Cons

- Have to manually add songs to library
